# 🌐 Test Environment – OpenCart Project

## 📌 Overview
Test environment is the setup where testing is performed, including hardware, software, network, and test data.

---

## 💻 Hardware
- Desktop / Laptop
- Mobile Devices (Android & iOS)

---

## 🖥 Software
- Browsers:
  - Google Chrome
  - Mozilla Firefox
  - Microsoft Edge
  - Safari

- Operating Systems:
  - Windows 10
  - macOS
  - Android
  - iOS

---

## 🌐 Network
- Stable Internet Connection
- Wi-Fi / Mobile Data
- Secure network with SSL support

---

## 🛠 Testing Tools
- Selenium (for automation)
- JIRA (Bug Tracking)
- Excel (Test Cases)
- Snipping Tool (Screenshots)

---

## 🗄 Test Data
- User Data (Login/Registration)
- Product Data
- Order Data

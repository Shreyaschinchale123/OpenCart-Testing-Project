# 🛒 OpenCart Testing Project

## 📌 Project Overview
This project focuses on testing an E-commerce application (OpenCart) including functionalities like login, registration, cart, checkout, and payment.

## 🎯 Objectives
- Validate functional requirements
- Ensure application usability & performance
- Identify and report defects

## 🔧 Testing Types
- Functional Testing
- Regression Testing
- Smoke & Sanity Testing
- UI Testing

## 📂 Project Documents
- Requirement Specification (SRS)
- Test Plan
- Test Cases
- Test Strategy
- Test Environment Setup

## 🧪 Tools Used
- JIRA (Bug Tracking)
- Excel (Test Cases)
- Manual Testing Techniques

## 🌐 Application Under Test
https://demo.opencart.com

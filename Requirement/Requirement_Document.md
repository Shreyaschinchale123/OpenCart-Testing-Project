# 📄 Requirement Document (SRS)

## 📌 Project Name:
OpenCart E-commerce Application

## 🎯 Objective:
To develop a secure and user-friendly online shopping platform where users can browse, purchase, and track orders.

---

## 🔑 Modules / Features

### 1. Login
- User should be able to login using email and password
- Successful login redirects to dashboard

### 2. Registration
- User can create a new account
- Required fields: Name, Email, Password

### 3. Add to Cart
- User can add products to cart
- Cart should display selected items

### 4. Checkout
- User can enter shipping & billing details
- User can place order successfully

### 5. Payment Gateway
- User can select payment method
- Payment should be secure and successful

### 6. Orders
- User can view order history
- Order details should be visible

---

## ✅ Acceptance Criteria Example (Login)
- User can open login page
- User enters valid credentials
- User clicks login button
- User redirected to dashboard
